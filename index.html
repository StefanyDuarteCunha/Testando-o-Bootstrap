Um pouco mais sobre o Bootstrap

O Bootstrap é um framework, com classes de CSS e, com apoio dele é possível criar sites responsivos e acessíveis, de forma mais rápida e eficiente.

No site https://getbootstrap.com/ é possível encontrar o tudo que é necessário para configurar o projeto com o Bootstrap.

Clicando em "Get Started" é possível acessar a documentação do Bootstrap, contendo os códigos utilizados para vinculação da ferramenta nos projetos, além de explicações para usar todos os benefícios ofertados pelo framework.
